Windows

To use this project on Windows, we need the WinUSB driver. Usually there shouldnt be any need to install drivers, however if theres issues connecting, or the ST-Link doesn't show up, you might want to try installing the driver manually.

1) Download Zadig.
2) Launch Zadig. If your ST-Link doesn't show up in the list, click on Options -> List all devices.
3) Select WinUSB next to the green arrow. Press "Replace Driver".
4) You should be good to go!
